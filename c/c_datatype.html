자료형 : 데이터를 표현하는 방법

정수형

  • char
    1바이트 = 28
    -128 ~ 127
  • short
    2바이트 = 216
    -32,768 ~ 32,767
  • int
    4바이트 = 232 -2,147,483,648 ~ 2,147,483,647
  • long
    4바이트 = 232
    -2,147,483,648 ~ 2,147,483,647
  • long long
    8바이트 = 264
    -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807
정수 자료형 앞에 unsigned를 붙이면 0이상의 정수만 표현됨
ex) unsigned int = 0 ~ (2,147,483,648 + 2,147,483,647)
