연산자

이항 연산자 : 두 개의 피연산자를 요구함

대입 연산자

=
연산자 오른쪽에 있는 값을 연산자 왼쪽에 있는 변수에 대입함

number = 10;

산술 연산자

+
두 피연산자의 값을 더함

number = 10 + 30;
-
왼쪽의 피연산자 값에서 오른쪽의 피연산자 값을 뺌

number = 20 - 10;
*
두 피연산자의 값을 곱함

number = 5 * 20;
/
왼쪽의 피연산자 값을 오른쪽의 피연산자 값으로 나눔

number = 100 / 5;
%
왼쪽의 피연산자 값을 오른쪽의 피연산자 값으로 나눴을 때의 나머지를 반환

number = 10 % 3;

증가, 감소 연산자 : 변수에 저장된 값을 1 증가, 감소 시킴

++
++number
number++
--
--number
number--
