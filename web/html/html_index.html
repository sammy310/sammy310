HTML

HTML(Hypertext Markup Language) : 웹페이지를 만들기 위한 언어

HTML은 크게 머리(HEAD), 몸통(BODY), 바닥(FOOTER)으로 나눠짐

머리(HEAD) : 웹 페이지에 대한 정보들을 정의

몸통(BODY) : 웹 페이지의 본문 내용

바닥(FOOTER) : 주로 저작권 정보나 서비스 제공자 정보등을 나타냄

HTML은 태그(tag) '<>'를 사용하여 작성

HTML은 대소문자 구분을 하지 않음

공백은 한 칸 이상 인식하지 않음

기본구조


<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>HTML</title>
</head>

<body>
    <h1>HTML에 대해서</h1>
    <p>HTML은 웹페이지를 만들기 위한 언어이다.</p>
</body>
</html>

설명

<HTML></HTML>
- HTML의 시작과 끝을 나타내는 태그
